Here are 50 terms that you may or may not know about Web Development!!!

Back end – This is the part of the website that is hidden from other users. This is where all the data, structure, applications and other content that builds up the site.

BETA - This is a live testing phase of a product or service.

Comments - Comments are used to make good reference and help designer and others on the functions of the code.

Cookies – This a small file that contains text and includes a unique identifier and information that is normally sent to a browser from a website. This data provides who has visited the site, their preferences and how often they do.

Compression - This is when an image or element source is reduce so it can reach to user quickly.

CSS – Also known as cascading style sheets. This is the code that web developers use to design how a web page should be displayed for the user. It formats too many things such as colour, background, links and many more.

Database - This is where multiple of data is stored in a way that can be manipulated.

Doctype- The doctype specifies which version of HTML is used in a document. It has a direct effect on whether your HTML will validate.

Domain – The specific name for a website that people use to search.

Error 404 - An error message indicating the server cannot find what was requested by the user.

Firewall - A system used to prevent access to a network or certain devices on a network.

Font - A typeface in one specific style and size.

Front end – this is the invert of the back end basically. This is what other can see when we looking at a website such as the content, page, images etc.

GIF - An image that is presented in a animation format

Graphical user interface – also known as GUI. This is involves the front end and its function is to allow you to interact the website without inputting any code to the site.

Graphics - This is where visual content is displayed.

HTML - Stands for Hypertext Markup Language. It’s the primary language used to write web pages.

HTTP - Also known as HyperText Transfer Protocol. This is a protocol where hypertext iMeta tag - is an HTML tag used to include meta data within the header of your web page.s moved between a browser and server.

Hyperlink - This where a link leads you to another web page by a click of the mouse.

Hypermedia - An extension to hypertext which includes graphics, audio, and video.

Internet - The main communication bewtween networks making the devices and computers connection possible

ISP - Internet Service Provider is a company that provides internet access.

JavaScript - An object-oriented high level language used for web development.

JPEG - This is is the best format type for an image/photo.

LAN - also stands for Local Area Network.

Markup - This refers to the coding applied to a text document to change it into an HTML, XML, or other Markup Language document.

Meta tag - is an HTML tag used to include meta data within the header of your web page.

Mp3 - This is an audio type where it is compressed and makes it easy transfer over the internet.

MySQL - A free open source database system commonly used for interaction with web applications.

Navigation - this is where users are able to manipulate through the site.

Platform - The central hardware and/or software of a computer.

PNG - Is a graphics format that uses lossless data compression

Protocol - A set of rules that chooses how programs, computers, and devices communicate with each other over the internet.

Resolution – this refers to the screen presentation such as the number of pixels. Higher the resolution the better it looks.

Router - A software or even hadware that moves data to devices to networks.

Script - this is a selective amount of code that makes the page more dynamic and interactive.

Selector - In CSS, the selector is the item a style will be applied to.

Specification - This is the requirements that the users wants or crucial data that is supposed to be there

Tag - A tag is a set of markup characters that are used around an element to indicate its start and end.

Template - this helps to keep a design consistent throughout the website.

URL - Also known as Uniform Resource Locator. This is the adress/link to find a specificc thing on the internet.

Usuability - This refers to the user using the site. Like how easy they find it to navigate around.

Valid - This is when web pages are those that return no errors based on the type of HTML. This is comonly checked though W3C.

Web Developer - This a person who specialises in designing and the construction websites to build many ther products/services.

Web Page - This is generally built in HTML and is referred to a single document. Websites are bulit from web pages.

Web server - Is a computer that has capabilities that allow it to host web sites and pages and make them available to internet users located elsewhere.

Web Site - This a specific location on the world wide web that ends with domain of .com, .co.uk, .net, .org .info.

Windows media - A set of audio and video formats developed by Microsoft.

Wireframe - a web design tool used to show representation of proposed functions, structure, and content of a website.

ZIP- A popular file compression format.
